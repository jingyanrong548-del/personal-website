# 合作伙伴 Logo 文件说明

请将以下公司的 Logo 图片文件放入此目录（`public/logos/`）：

## 需要的 Logo 文件

1. **basf-logo.png** - BASF（巴斯夫）
2. **dow-logo.png** - Dow Chemical（陶氏化学）
3. **petrochina-logo.png** - PetroChina（中石油）
4. **sinopec-logo.png** - Sinopec（中石化）
5. **cas-logo.png** - CAS（中科院）
6. **shuanghui-logo.png** - 双汇集团
7. **gcl-logo.png** - 协鑫集团
8. **longxiang-logo.png** - 龙翔实业
9. **pg-logo.png** - P&G（宝洁公司）

## Logo 要求

- **格式**：PNG（推荐，支持透明背景）或 SVG
- **尺寸**：建议宽度 200-300px，高度自适应
- **背景**：透明背景或白色背景
- **颜色**：可以是彩色或单色，CSS会自动应用灰度滤镜（悬停时恢复彩色）

## 注意事项

- 确保 Logo 文件名称与上述列表完全一致
- Logo 会自动适配容器大小
- 默认显示为灰度，悬停时恢复彩色
- 如果某个 Logo 暂时无法获取，可以先使用占位符图片
